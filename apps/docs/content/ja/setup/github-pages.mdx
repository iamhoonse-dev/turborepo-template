# 📖 GitHub Pages

このプロジェクトはGitHub Pagesを使って <LocalizedLink href="apps/sample-apps/docs">`docs`</LocalizedLink> アプリを公開しています。
GitHub Pagesで正常に公開するには、SettingsでPagesを有効化する必要があります。

## GitHub Pages有効化手順

1. GitHubリポジトリのページに移動します。
2. `Settings` タブをクリックします。
3. `Pages` セクションに移動します。
4. `Build and deployment` で `Source` を `GitHub Actions` に設定します。
