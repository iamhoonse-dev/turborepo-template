# 🏷 リポジトリ権限の設定

##

changesetsボットを利用するには、GitHubリポジトリの設定を変更する必要があります。
この設定により、ChangesetsボットがPRを作成し、変更内容を自動で追跡できるようになります。
以下の手順に従ってください：

## リポジトリ権限の設定方法

1. GitHubリポジトリのページに移動します。
2. `Settings` タブをクリックします。
3. `Actions` セクションに移動します。
4. `General` を選択します。
5. Workflow Permissionsで `Read repository contents permission` を `Read and write permissions` に変更します。
6. `Allow GitHub Actions to create and approve pull requests` をチェックします。
7. `Save` ボタンをクリックします。
