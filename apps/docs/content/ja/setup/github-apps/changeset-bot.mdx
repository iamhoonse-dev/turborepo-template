# 🦋 changeset-bot

このプロジェクトはChangesetを使って各npm公開対象パッケージのバージョンを管理し、GitHub Actionsを通じて自動でリリースを作成するように設定されています。
Changesetはパッケージの変更内容を記録し、それに基づいてバージョンを更新し、リリースノートを生成するツールです。
Changeset BotはGitHubでChangesetを利用した自動化リリースプロセスをサポートするGitHub Appです。

## changeset-bot GitHub Appのインストール

Changeset Bot GitHub Appをインストールするには、以下の手順に従ってください：

1. [Changeset Bot App](https://github.com/apps/changeset-bot) ページにアクセスします。
2. `Install` ボタンをクリックします。
3. インストールするリポジトリを選択します。
4. `Install and Authorize` ボタンをクリックします。
