# 🚀 クイックスタート

## このテンプレートを使う

「Use this template」ボタンをクリックしてリポジトリを作成した後、以下のコマンドを実行してください。

## 依存関係のインストール

このプロジェクトで提供されるモノレポコマンドを使うには、まず依存関係をインストールする必要があります。以下のコマンドを実行してください：

```bash
pnpm install
```

## オーナーIDの変更

コードベース内のすべてのファイルを検索し、既存のオーナーIDが記載されている部分を変更します。
詳細は[`tools/cli`のREADME](./tools/cli/README.md#-提供機能)を参照してください。

```bash
change-owner-name --name your-username
```

## リポジトリ名の変更

コードベース内のすべてのファイルを検索し、既存のリポジトリ名が記載されている部分を変更します。
詳細は[`tools/cli`のREADME](./tools/cli/README.md#-提供機能)を参照してください。

```bash
rename-repository --name new-repository-name
```

## スコープ名の変更

コードベース内のすべてのファイルを検索し、既存のスコープ名が記載されている部分を変更します。
モノレポ内の一部コンポーネントを[npm](https://www.npmjs.com/)の特定スコープ（例：`new-scope-name`）で公開したい場合に便利です。
詳細は[`tools/cli`のREADME](./tools/cli/README.md#-提供機能)を参照してください。

```bash
rename-scope --name new-scope-name
```

## 開発サーバーの起動

これで開発サーバーを起動してアプリケーションを確認できます。以下のコマンドを実行してください：

```bash
pnpm dev
```
