This project uses Changeset to manage the version of each npm-distributable package and is configured to automatically create releases via GitHub Actions.
Changeset is a tool that records changes to packages, updates versions based on those changes, and generates release notes.
The Changeset Bot is a GitHub App that supports automated release processes using Changeset on GitHub.

## Installing the changeset-bot GitHub App

To install the Changeset GitHub App, follow these steps:

1. Go to the [Changeset Bot App](https://github.com/apps/changeset-bot) page.
2. Click the `Install` button.
3. Select the repository to install it on.
4. Click the `Install and Authorize` button.
