This project deploys the [`docs`](./apps/docs) app using GitHub Pages.
To ensure GitHub Pages deployment works properly, you need to enable Pages in the repository Settings.

## How to Enable GitHub Pages

1. Go to your GitHub repository page.
2. Click the `Settings` tab.
3. Go to the `Pages` section.
4. In `Build and deployment`, set `Source` to `GitHub Actions`
