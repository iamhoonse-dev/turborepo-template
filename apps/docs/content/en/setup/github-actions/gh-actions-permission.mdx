##

To use the changesets bot, you need to change your GitHub repository settings.
This allows the Changesets bot to create PRs and automatically track changes.
Please follow the steps below:

## How to Set Repository Permissions

1. Go to your GitHub repository page.
2. Click the `Settings` tab.
3. Go to the `Actions` section.
4. Select `General`.
5. In Workflow Permissions, change `Read repository contents permission` to `Read and write permissions`.
6. Check `Allow GitHub Actions to create and approve pull requests`.
7. Click the `Save` button.
