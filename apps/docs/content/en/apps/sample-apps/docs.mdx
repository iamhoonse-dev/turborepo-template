The `apps/docs` workspace is a Next.js-based application responsible for documentation (Docs) within the monorepo.
It leverages Nextra and the Nextra Docs theme to help you build fast and efficient documentation sites.
It provides documentation, guides, and examples for various packages, modules, and components in the project, making it easy for both developers and users to find information.

## ğŸš€ Nextra-based Documentation System

This app is built on **Nextra** and the **Nextra Docs theme**. Nextra is a powerful documentation site framework running on Next.js, offering the following advantages:

### Benefits of Next.js

- **Based on Next.js**: Uses the latest Next.js framework to provide fast and efficient static/dynamic documentation pages.
- **MDX Support**: Supports the MDX format, combining Markdown and JSX, so you can use React components within your docs.
- **Monorepo Integration**: Integrated with other apps/packages in the Turborepo environment, making it easy to reference shared components and utilities.
- **Custom Components**: Define custom components for use in docs via `mdx-components.ts`.

### Benefits of Nextra

- **Rapid Documentation**: Easily build documentation sites using just folder structure and MDX files.
- **Theme Support**: Offers stylish documentation themes and various customization options via `nextra-theme-docs`.
- **Built-in Search**: Integrates with Pagefind for fast full-document search even after static builds.
- **Versioning & i18n**: Easily manage documentation versions and multiple languages via Nextra settings.

    - `en/` : English docs

- `public/` : Static files (images, fonts, etc.)
