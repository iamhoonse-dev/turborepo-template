# ▲ 새 Next 앱 추가하기

## 📋 복사하여 추가하기

1. 터미널에서 프로젝트 루트로 이동한 뒤, `apps/empty-next-app` 폴더를 복사해 새 앱 폴더를 만듭니다.

   예시:
   ```bash
   cp -R apps/empty-next-app apps/새로운앱이름
   ```

2. 복사한 폴더의 이름과 내부의 `package.json` 등에서 앱 이름을 원하는 이름으로 변경합니다.

3. 패키지 설치:
   ```bash
   pnpm install
   ```

4. 앱 실행:
   ```bash
   pnpm --filter 새로운앱이름 dev
   ```

## 🛠️ 추가 설정

템플릿(`apps/empty-next-app`)은 최소 설정만 포함되어 있으니, 프로젝트에 맞게 자유롭게 커스터마이즈해서 사용해 주세요.

- 필요에 따라 `next.config.js`, `tsconfig.json` 등 설정 파일을 수정하세요.
- 모노레포 프로젝트 내 다른 앱과의 의존성을 추가하려면 <LocalizedLink href="/packages/using-in-other-workspaces">`패키지 > 패키지 사용하기`</LocalizedLink> 문서를 참고하세요.
