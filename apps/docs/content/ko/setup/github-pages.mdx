# 📖 GitHub Pages

이 프로젝트는 GitHub Pages를 사용해서 <LocalizedLink href="apps/sample-apps/docs">`docs`</LocalizedLink> 앱을 배포하고 있어요.
GitHub Pages 배포가 정상적으로 동작하려면 Settings에서 Pages를 활성화해야 해요.

## GitHub Pages 활성화 방법

1. GitHub 레포지토리 페이지로 이동해요.
2. `Settings` 탭을 클릭해요.
3. `Pages` 섹션으로 가요.
4. `Build and deployment`에서 `Source`를 `GitHub Actions`로 설정해요.
