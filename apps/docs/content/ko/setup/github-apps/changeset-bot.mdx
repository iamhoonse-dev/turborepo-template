# 🦋 changeset-bot

이 프로젝트는 Changeset 을 사용하여 각 npm 배포 대상 패키지의 버전을 관리하고, GitHub Actions 를 통해 자동으로 릴리스를 생성하도록 설정되어 있어요.
Changeset 은 패키지의 변경 사항을 기록하고, 이를 기반으로 버전을 업데이트하며, 릴리스 노트를 생성하는 도구예요.
Changeset Bot 은 GitHub 에서 Changeset 을 이용한 자동화된 릴리스 프로세스를 지원하는 GitHub App 이에요.

## changeset-bot GitHub App 설치

Giscus GitHub App을 설치하려면 다음 단계를 따라 주세요:

1. [Changeset Bot App](https://github.com/apps/changeset-bot) 페이지로 이동해요.
2. `Install` 버튼을 클릭해요.
3. 설치할 레포지토리를 선택해요.
4. `Install and Authorize` 버튼을 클릭해요.
