# 🤖 GitHub Actions

이 프로젝트는 GitHub Actions를 활용하여 자동화된 CI/CD 파이프라인을 구축하였습니다.
코드 품질 검증, 테스트, 빌드, 배포 등 주요 개발 및 운영 과정을 자동화함으로써 개발 효율성을 높이고, 코드 품질을 유지합니다.

## 주요 워크플로우

- **Test**
  코드가 push될 때마다 자동으로 실행되어, 유닛 테스트, E2E 테스트, Storybook 테스트 등 다양한 검증 과정을 수행합니다.

- **Release**
  `main` 브랜치에 변경이 감지되면 실행되며, 패키지 버전 관리 및 Changesets 기반 릴리즈 PR 생성을 자동화합니다.

- **Publish Github Pages**
  문서 사이트를 정적으로 빌드하여 `gh-pages` 브랜치로 배포, GitHub Pages를 통해 최신 문서를 자동으로 제공합니다.
