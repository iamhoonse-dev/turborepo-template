# 🟦 TypeScript 설정

`configs/typescript-config` 워크스페이스는 모노레포 내 여러 패키지와 앱에서 공통으로 사용할 수 있는 TypeScript 설정(공유 tsconfig)을 제공합니다.

## 주요 목적

- 프로젝트 전반에 걸쳐 일관된 TypeScript 컴파일 옵션과 코드 스타일을 유지합니다.
- 각 워크스페이스(앱/패키지)에서 이 설정을 확장하거나 그대로 사용할 수 있습니다.

## 주요 구성

- **`base.json`**\
  모든 패키지/앱에 공통 적용되는 기본 TypeScript 옵션을 정의합니다.
- **`react-library.json`**\
  React 라이브러리 개발에 특화된 추가 옵션을 포함합니다.
- **`nextjs.json`**\
  Next.js 프로젝트에 특화된 추가 옵션을 포함합니다.

## 사용 방법

### 상속하기

각 워크스페이스의 `tsconfig.json`에서 `@repo/typescript-config/{config명}`을 extends로 지정하여 사용할 수 있습니다.

```jsonc filename="apps/docs/tsconfig.json"
{
  "extends": "@repo/typescript-config/nextjs.json",
  "compilerOptions": {
    // 워크스페이스별 추가/재정의 옵션
  }
}
```

## 참고

- 실제 옵션 및 상세 설정은 `configs/typescript-config` 워크스페이스의 각 config 파일을 참고하세요.
- 새로운 공통 옵션이 필요하다면 이 워크스페이스에 추가하여 전체 프로젝트에 적용할 수 있습니다.
