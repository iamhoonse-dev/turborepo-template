# 🧩 공통 설정 구성

`configs` 디렉토리는 모노레포 전역에서 공유할 공통 설정 구성을 정의합니다.

## 🎯 목적

이렇게 공통 설정을 정의함으로써, 모노레포 내의 모든 프로젝트가 일관된 개발 환경을 유지할 수 있습니다.
이는 코드 품질을 높이고, 개발자 경험을 향상시키며, 유지보수를 용이하게 합니다.

## 📁 주요 구성

이 디렉토리에는 ESLint, TypeScript, Vitest, Playwright 등과 같은 도구의 설정 파일이 포함되어 있습니다.
