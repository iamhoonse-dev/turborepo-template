# 📦 npm 배포 활성화

## release 워크플로우에서 npm 배포 활성화하기

release 워크플로우에서 npm 배포는 기본적으로 활성화 되어 있지 않아요. \
npm 배포를 활성화하려면, <LocalizedLink href="/setup/npm">`설정 > NPM`</LocalizedLink> 문서를 참고하여

- NPM 토큰을 설정하고,
- 워크플로우 내에서 npm 배포를 실행하도록 활성화해야 해요.

## `package.json` 파일에서 npm 배포 설정하기

npm 배포를 위해서는 각 배포 대상 패키지의 `package.json` 파일에서 `publishConfig` 필드 및 `private` 필드를 설정해야 해요.

- `publishConfig` 필드는 `access` 속성을 `public`로 설정해야 하며,
- `private` 필드는 `false`로 설정해야 해요.

```json filename="package.json"
{
  "private": false,
  "publishConfig": {
    "access": "public"
  },
  // ...
}
```
